Enhanced FAA CIFP Data

Welcome to the FAA Coded Instrument Flight Procedures (CIFP) data enhancer. It amends the FAA's CIFP data to fix localizer bearings to work better in flight simulators like X-Plane. See this blog post for more information.

Instructions

  1. Right-click the download link for the cycle you want and save the file to your computer.
  2. Rename the file from FAACIFP18_processed_MM-DD-YYYY.txt to earth_424.dat.
  3. Copy the above file to your X-Plane "Custom Data" directory. A few example locations for this directory are listed below if you use Steam; your exact path to the directory may differ. See the official X-Plane instructions for more information.
    Windows: C:\Program Files (x86)\Steam\steamapps\common\X-Plane 11\Custom Data
    Mac: "${HOME}/Library/Application\ Support/Steam/steamapps/common/X-Plane\ 11/Custom\ Data/earth_424.dat"
